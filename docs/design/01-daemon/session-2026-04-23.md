### Daemon code migration

**`crates/senseid/src/db/` needs migration from SQLite to PostgreSQL:**

| File | Current state | Needed |
|------|--------------|--------|
| `store.rs` | SQLite schema + queries for repos, sessions, tags, events, etc. | Rewrite for PostgreSQL using sqlx |
| `mod.rs` | SQLite connection pool setup | PostgreSQL connection pool (sqlx::PgPool) |

**Specific changes:**
- All `CREATE TABLE` in store.rs → use DDL files (already written)
- All `rusqlite` queries → `sqlx` async queries against PostgreSQL
- JSON text columns → proper `jsonb` columns
- TEXT arrays → PostgreSQL `text[]` arrays
- sqlite-vec → pgvector (HNSW already in DDLs)
- FTS5 → PostgreSQL full-text search (tsvector/tsquery)
